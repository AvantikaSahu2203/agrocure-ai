# AgroCure AI

AgroCure AI is an AI-powered crop disease detection and medicine ecosystem designed to assist farmers, admins, and store owners.

## Project Structure

- **backend/**: FastAPI backend forAPI services, database interactions, and AI integration.
- **web/**: Next.js frontend for Farmer, Admin, and Store dashboards.
- **mobile/**: React Native mobile application for farmers.
- **ai_agent/**: Python scripts and modules for AI model training and inference.

## Getting Started

### Prerequisites

- Node.js (v18+)
- Python (v3.9+)
- Docker (optional, recommended for database)
- PostgreSQL
